## The Philosophy

> "I choose a lazy person to do a hard job. Because a lazy person will find an easy way to do it." — Bill Gates

I'm not "building in public" for clout. I'm doing it because:

1. **Accountability** — if I say I shipped something, I have to actually ship it
2. **Documentation** — future-me will thank me
3. **Serendipity** — you never know who reads this stuff
